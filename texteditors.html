HTML Notes

The number after the h tag determines the size that the text will be displayed starting from 1 and ending with 6. h tags are generally only used for headings and subheadings

To change the color of the text and font size you can link certain text to your css folder with a span tag to stay inline. You can use span to stay in line and div, to create a new line.

Adding the strong tag creates a bolder text You can even use it within a paragraph to bold specific words, another inline text editor like the strong tag you can use is em to Emphasize text inline.

To add notes and comments on your code that will not show up on the web page use *!-- replacing the * for a < --*
